# Agent-Based Modeling of Endogenous Systemic Risk

Repository containing the simulation engine and codebase for **Flori Kusari's Bachelor Thesis (2026)** at Constructor University, under the supervision of Prof. Dr. Sören Petrat.

This project implements a continuous-time Agent-Based Model (ABM) to study price-mediated contagion, fire sales, and modern high-velocity digital bank runs across varying financial network topologies.

## 📂 Project Structure

* `Scale_Free_Graph_Thesis_ABM.ipynb` — Main simulation suite execution running on a Barabási-Albert scale-free network architecture.
* `SBM_Graph_Thesis_ABM.ipynb` — Comparative baseline simulation running on a Stochastic Block Model (SBM) network architecture.
